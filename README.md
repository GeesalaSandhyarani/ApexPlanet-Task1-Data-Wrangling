# Task 1 - Data Immersion & Wrangling

## Objective

The objective of this task was to understand, clean, and prepare a sales dataset for analysis.

## Dataset Overview

The dataset contains customer information, product details, and sales records.

## Tasks Performed

* Loaded dataset using Pandas
* Explored dataset using head(), info(), and describe()
* Identified missing values
* Checked duplicate records
* Calculated Mean, Median, and Mode
* Filled missing Age values using Median
* Filled missing City values using Mode
* Saved cleaned dataset
